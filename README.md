# 商城管理系统 java106

## 项目概述

商城管理系统是一个基于SpringBoot和Vue的前后端分离的网上商城系统，适用于销售电脑、手机、服装、数码产品等多类型商品。

## 技术栈

- 后端：Java，Spring Boot，MyBatis，Redis，Shiro
- 前端：Vue，Axios，Element UI，Vuex
- 运行环境：JDK 8，MySQL 8.2.1，IDEA，Maven，Redis，Node.js 14，VSCode

## 功能模块

### 前台模块

- 用户登录
- 用户注册
- 购物车
- 我的订单
- 个人中心
- 商品评价
- 修改密码

### 后台模块

- 管理员登录
- 订单管理
- 商品管理
- 营销管理
- 权限管理
- 个人中心

## 系统角色

- 普通用户
- 管理员

## 运行环境

- JDK 1.8 或更高版本
- MySQL 8.2.1 或更高版本
- IDEA 或其他 Java IDE
- Maven
- Redis
- Node.js 14 或更高版本
- VSCode 或其他代码编辑器

## 目录结构

```
 ├── front-end        // 前端代码
 ├── back-end         // 后端代码
 ├── sql              // 数据库脚本
 └── doc              // 项目文档
```

## 核心亮点

- 前后端分离，易于维护和扩展
- 采用分布式缓存Redis，提高系统性能
- 集成权限管理框架Shiro，保障系统安全
- 提供丰富的功能模块，满足多种业务需求
- 灵活的可配置性，适应不同场景下的应用
